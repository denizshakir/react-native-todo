# React Native Todo app

A simple todo list application built with react-native.

## Getting started

### Clone repository
```
git clone https://github.com/goodfellas7/react-native-todo.git
```

### Installation
#### Expo cli
```
npm install -g expo-cli
```
#### Npm packages
```
cd Todo
npm install
```

### Running
#### For android, run
```
npm run android
```
#### For iOS, run
```
npm run ios
```

## Technology Stack
- expo
- react-native
- react-navigation
- redux
